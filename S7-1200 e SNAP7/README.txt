Instalar Python 3
Instalar o snap7 - pip install python-snap7

Copiar o arquivo snap7.dll que está em snap7-full-1.4.0\release\Windows\Win64
Colar o arquivo na pasta System32 do Windows
Pasta  snap7-full-1.4.0 foi baixada em https://sourceforge.net/projects/snap7/files/1.4.0/

Programa do CLP desenvolvido no TIA V15.1 update 3 para uma CPU S7-1200 

Para que o snap7 consiga acessar o CLP é necessário fazer a configuração mostrada na imagem foto1
Os DBs não podem estar com acesso otimizado, como mostra a imagem foto2.

Site com a documentação e dicas de como utilizar o snap7:
http://snap7.sourceforge.net/sharp7.html
https://python-snap7.readthedocs.io/en/latest/introduction.html

Obs: o snap7 está presente em várias linguagens (C#, C++, VBScript, .NET e Pascal (Delphi))
